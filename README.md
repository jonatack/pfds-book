# Pfds-Book

A translation (in progress) of data structures from the book
Purely Functional Data Structures by Chris Okasaki.

## Usage

```common-lisp
(ql:quickload :pfds-book)
```

## Installation

Clone repo into ~/quicklisp/local-projects.
